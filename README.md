# Tips

使用bat文件运行哦 每个部分都有提示哦~

本程序自动获取剪贴板中链接的ID 并生成点歌指令

复制完分享链接 直接点击获取就好啦

会把复制过的ID存储下来哦~

不复制链接会随机调取哦~
